# DARWIS

DARWIS is a macOS AI interrupt assistant for asking questions about the current moment — what you see on screen, what you just heard, or both.

Press a shortcut, ask a question, and get a concise answer without copying text, taking screenshots, rewinding audio, or explaining context manually.

## Core idea

DARWIS helps users stay in flow by capturing just enough real-time context to answer quick questions.

## MVP focus

- Global hotkey invocation
- Voice question input
- Screen context capture
- Recent audio context
- Short answer overlay
- Privacy-first context boundaries

## Status

Early product planning and prototyping stage.
